# Проект "API_final_yatube"

## Описание

Проект включает в себя функционал для обмена иформации с приложением **Yatube** через API, которое позволяет создавать различные публикации, комментарии к ним, где также можно оформить подписку и отписку от автора той или иной публикации.

Из проекта исключен фронтенд.

Аутентифицированным пользователям разрешено изменение и удаление своего контента; в остальных случаях доступ предоставляется только для чтения.

Когда вы запустите проект, по адресу [Redoc](http://127.0.0.1:8000/redoc/) будет доступна документация для API Yatube. В документации описано, как работает API. Документация представлена в формате **Redoc**.

## Установка

Клонировать репозиторий и перейти в него в командной строке:

```text
git clone https://github.com/Orion-87/api_final_yatube.git
```

```text
cd api_final_yatube
```

Cоздать и активировать виртуальное окружение:

```text
python -m venv venv
```

```text
source venv/Scripts/activate
```

Установить зависимости из файла requirements.txt:

```text
python -m pip install --upgrade pip
```

```text
pip install -r requirements.txt
```

Выполнить миграции:

```text
cd yatube_api
```

```text
python manage.py migrate
```

Запустить проект:

```text
python manage.py runserver
```

## Примеры. Некоторые примеры запросов к API

### Получение комментария

Получение всех комментариев к публикации.

```text
http://127.0.0.1:8000/api/v1/posts/{post_id}/comments/
```

Ответ:

```text
{
"id": 0,
"author": "string",
"text": "string",
"created": "2019-08-24T14:15:22Z",
"post": 0
}
```

Более подробно запросы можно посмотреть по адресу [Redoc](http://127.0.0.1:8000/redoc/).
