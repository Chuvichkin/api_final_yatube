# Проект "API_final_yatube"

## Описание

Проект включает в себя функционал для обмена информации с приложением **Yatube** через API, которое позволяет создавать различные публикации, комментарии к ним, где также можно оформить подписку и отписку от автора той или иной публикации.

Из проекта исключен фронтенд и view-функции приложения posts.

Аутентифицированным пользователям разрешено изменение и удаление своего контента, в остальных случаях доступ предоставляется только для чтения.

Когда вы запустите проект, по адресу [Redoc](http://127.0.0.1:8000/redoc/) будет доступна документация для API Yatube. В ней приведено описание функционирования API. Документация в формате **Redoc**.

## Установка

Клонировать репозиторий и перейти в него в командной строке:

```text
git clone git@github.com:Chuvichkin/api_final_yatube.git
```

```text
cd api_final_yatube
```

Cоздать и активировать виртуальное окружение:

```text
python -m venv venv
```

```text
source venv/Scripts/activate
```

Установить/обновить pip и зависимости из файла requirements.txt:

```text
python -m pip install --upgrade pip
```

```text
pip install -r requirements.txt
```

Перейти в директорию с manage.py и произвести миграции:

```text
cd yatube_api
```

```text
python manage.py migrate
```

Запустить сервер:

```text
python manage.py runserver
```

## Примеры запросов к API.

### Получение комментария

Получение всех комментариев к публикации.

```text
http://127.0.0.1:8000/api/v1/posts/{post_id}/comments/
```

Ответ:

```text
{
"id": 0,
"author": "string",
"text": "string",
"created": "2019-08-24T14:15:22Z",
"post": 0
}
```

Полный список запросов расположен по адресу [Redoc](http://127.0.0.1:8000/redoc/).
